Labour — slab m² bands

The man-days a flat ground slab takes, by area. Used when a pour's labour is set to "Auto — by slab m²". Editable, and each shows its default so you can revert. Above the top band: base days + one day per "per extra m²".

Every pour's labour in man-days and dollars — the base labour (m² band, % of materials, or manual) plus each section's extra crew, excavation and cantilever days. Man-day rate is set under Rates & labour.
